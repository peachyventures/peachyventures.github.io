Sonoros v1.7

We've been working hard to bring you exciting new features & squash some pesky bugs in this release.

New in v1.7

- We've shrunk spacing around grid tickets to fit more of them on screen.
- Added license remaining information to the licensing screen.
- Added the ability to change the IP addresses of your devices.
- Fixed a bug that led to the 'Set all principles' button not to work in the players view.
- Fixed a bug that led to some icons drawing incorrectly.
- Other bug fixes & improvements.
