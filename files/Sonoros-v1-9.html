Sonoros v1.9

We've been working hard to bring you exciting new features & squash pesky bugs in this release.

New in v1.9

- Improvements to exporting audio including option to export all or multiple channels.
- Arrow keys now always change active listen while on the monitoring view.
- Chat text box always stays focused while chat is open to speed up sending messages.
- Inpector height and visible state now persist between launches.
- Added demo licensing.
- Fix to an issue that could cause long ticket titles to clip.
- Improved players layout in inspector.
- Other bug fixes and improvements.


If you spot an issue or have an idea for a feature, get in touch at support@peachy.ventures.
